Three Principles

제가 일할 때 지키는 세 가지 원칙

방향을 먼저 정하고, 경험을 다음 업무의 기준으로 남기며, 빈 화면에서 시작하지 않습니다.

01

Start with Why

모든 일은 왜 해야 하는지부터 시작합니다. 방향이 정해지면 최종 결과물이 빠르게 그려지고, 그다음부터는 하나씩 살을 붙여갑니다.

문제를 먼저 정의합니다.

목적에 맞는 결과물을 그립니다.

결정한 방향을 끝까지 유지합니다.

02

Never Build Twice

처음 하는 일은 시간이 걸릴 수 있습니다. 대신 다음에는 더 빠르고 더 나은 방식으로 할 수 있도록 기준과 레슨런을 남깁니다.

반복 가능한 구조를 만듭니다.

배운 점과 개선점을 기록합니다.

누구나 이어갈 수 있도록 정리합니다.

03

Don’t Start from Scratch

완전히 새로운 일은 생각보다 많지 않습니다. 기존 경험과 레퍼런스, AI를 활용해 빠르게 시작하고 더 중요한 판단과 완성도에 집중합니다.

기존 경험을 적극 활용합니다.

레퍼런스와 AI로 빠르게 시작합니다.

맥락과 목적에 맞게 완성합니다.
